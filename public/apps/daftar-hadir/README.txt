Sistem Daftar Hadir GreenroomID

1. Data diproses lokal di browser dan tidak disimpan ke database GreenroomID.
2. Database hanya menerima hitungan penggunaan saat pengguna memilih PDF atau Print.
3. Export yang aktif hanya PDF / Save as PDF dan Print.
4. Export Word dan Excel dinonaktifkan.
5. Tanda tangan hanya muncul setelah akhir tabel pada halaman terakhir.
6. Teks kecil https://greenroomid.com ditampilkan di kiri atas dokumen sebagai penanda ringan.
